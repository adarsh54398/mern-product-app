# 🚀 MERN Product App

A simple full-stack MERN application to manage products.

## 🛠️ Tech Stack
- ⚛️ React (Frontend)
- 🟢 Node.js & Express (Backend)
- 🗄️ MongoDB (optional / in-memory storage)

## ✨ Features
- ➕ Add Product
- 📋 View Products
- 🔗 API Integration (Frontend ↔ Backend)
- ⚛️ Component-based UI using props
